Quantifier Pro

Quantifier Pro es un plugin para SketchUp que permite la realización de mediciones y cuantificaciones directamente desde el entorno de SketchUp generando listados de salida de las mediciones así como de los precios de cada una de las unidades de medición. Permite la medición y cuantificación por materiles, por etiquetas, por componentes, etc.

Rtches Bim Utilities

Bim Utilities consiste en una serie de iconos que nos permiten añadir información a nuestro modelos para permitir una exportación del modelo a formato de intercambio estandard IFC de una forma sencilla.
Además podemos enriquecer nuestro modelo añadiendo cualquier tipo de parámetro personalizado y extraer tablas para incluirlas en nuestros planos o entrgables.

OpenCutList

Open Cut List es un plugin que permite realizar despieces de elementos modelados separando las piezxas de las que consta de forma gráfica e incluso permitiendo hacer mediciones de las piezas y extraer los planos de corte para maquinas de corte mecanizado o para el corte de tableros de madera entre otros.
