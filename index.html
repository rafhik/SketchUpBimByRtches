Arquitectura

Profile Builder

Descripción. Profile Builder es un plugin que permite dibujar cualquier sección que se nos ocurra a lo largo de una alineación. Permite crear muros, pilares, barandillas, vigas, zapatas, etc. Además permite configurar conjuntos de elementos que se dibujan a lo largo de una alineación de forma paramétrica.

FlexTools

Descripción. FlexTools permite dibujar de una forma rápida elementos paramétricos como puertas, ventanas, escaleras, rampas, etc. Además dispone de distintas funcionalidades que ahorran tiempo a la hora de gestionar bibliotecas de componentes.

Rtches Bim Utilities

Bim Utilities consiste en una serie de iconos que nos permiten añadir información a nuestro modelos para permitir una exportación del modelo a formato de intercambio estandard IFC de una forma sencilla.
Además podemos enriquecer nuestro modelo añadiendo cualquier tipo de parámetro personalizado y extraer tablas para incluirlas en nuestros planos o entrgables.

Enlace. SketchUcation. Pluginstore
Compra en España. Gratuito en la página de descarga

Modelado paramétrico. Componentes dinámicos

Los componentes dinámicos son una funcionalidad incluida en SketchUp que nos permite la creación de componentes con distintos parámetros en funciona de los cuales la geometría adopta distintos tamaños o combinadiones de objetos.

Enlace. Extension Warehouse.
Compra en España. Gratuito incluido en SketchUp Pro

Polycam

Polycam es un plugin que sirve de complemento a una aplicación para dispositivos móviles para escaneo de nubes de puntos que permite la descarga de los escaneos realizados directamente a SketchUp sin la necesidad de utilizar herramientas de tratamiento de nubes de puntos.

Enlace. Extension Warehouse.
Compra en España. Gratuito para SketchUp Pro

Scan Essentials (SketchUp Studio)

Scan Essentials es un plugin que forma parte de las suscripciones de SketchUp en su versión Studio que sirve para trabajar con nubes de puntos directamente desde SketchUp. Permite cargar las nubes de puntos y tomarlas como referencia para realizar el modelado de forma más precisa.

Enlace. Incluido en las versiones de suscripción de SketchUp Pro

3Skeng

3Skeng es un plugin muy completo de modelado profesional de tuberías, conductos, soportes y demas elementos de instalaciones MEP. Su uso es sencillo, intutivo y muy práctico y permite el modelado de instalaciones con un grado de detalle muy elevado. Tambien permite el etiquetado de los modelos, la extracción de mediciones y listados de componentes así como su exportación a IFC para coordinación o para cálculo de trazados.
