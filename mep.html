3Skeng

3Skeng es un plugin muy completo de modelado profesional de tuberías, conductos, soportes y demas elementos de instalaciones MEP. Su uso es sencillo, intutivo y muy práctico y permite el modelado de instalaciones con un grado de detalle muy elevado. Tambien permite el etiquetado de los modelos, la extracción de mediciones y listados de componentes así como su exportación a IFC para coordinación o para cálculo de trazados.

VBO Piping

VBO Piping Pro es un plugin que permite la creación de trazados de tuberías y conductos a partir de alineaciones ya creadas en el modelo. Requiere la generación de componentes para cada tipología de tubería o conducto así como la configuración de cada una de sus uniones.

Profile Builder

Profile Builder es un plugin que nos permite extrusionar cualquier trazado lineal de tuberías o conductos, solo es necesario generar el perfil de la sección a extruir que será un sencillo círculo para las tuberías y un rectángulo para los conductos. Se pueden realizar canalizaciones eléctricas y bandejas mediante el uso de Assemblies o conjuntos de componentes y trazados.
