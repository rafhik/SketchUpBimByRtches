Planificación 4D

Skp4DPro

Skp4DPro es un plugin para SketchUp que permite enlazar los elementos del modelo con una planificación de Microsoft Project y visualizar la planificación junto con el modelo e incluso realizar el seguimiento de las actividades completadas y las actividades pendientes
